# Як знайти Node IDs для зображень у Figma

## Швидкий спосіб через URL

1. Відкрийте дизайн: https://www.figma.com/design/GgvThj8CrZbvXq4beuHs1p/Shopping-Ecommerce-Website-UI--Community-?node-id=3225-5596

2. Виберіть елемент з зображенням (наприклад, hero image)

3. Подивіться на URL браузера - там буде параметр `node-id=XXXX-YYYY`

4. Node ID буде у форматі `XXXX:YYYY` (замініть дефіс на двокрапку)

## Детальний спосіб

### Для Hero Image:
1. Виберіть зображення жінки в червоному пальті (права частина hero секції)
2. Node ID буде в URL або в DevTools

### Для Categories:
1. Виберіть кожне зображення категорії окремо
2. Запишіть Node ID для кожного:
   - Casual Wear (чоловік в червоній сорочці)
   - Women Top (жінка в кремовій блузці)
   - Ethnic Wear (жінка в помаранчевому топі)
   - Kids Wear (дитина в світло-блакитному)

### Для Products:
1. Виберіть кожен продукт окремо в сітці "Our Bestseller"
2. Запишіть Node ID для кожного продукту

### Для Deals Image:
1. Виберіть зображення жінки в чорному пальті (права частина Deals секції)

### Для Instagram:
1. Виберіть кожне з 4 зображень окремо

### Для Testimonials:
1. Виберіть кожен аватар окремо

## Оновлення скрипта

Після того, як ви знайдете Node IDs, оновіть файл `scripts/export-figma-images.js`:

```javascript
const imageMapping = {
  'hero-image.jpg': '3225:5596', // Ваш знайдений Node ID
  'casual-wear.jpg': 'XXXX:YYYY', // Ваш знайдений Node ID
  // ... і так далі
};
```

## Альтернативний спосіб - через Figma UI

Якщо не хочете використовувати API:

1. Відкрийте Figma
2. Виберіть елемент
3. У правій панелі натисніть "Export"
4. Виберіть формат (PNG/JPG) та scale (2x)
5. Натисніть "Export" і збережіть файл
6. Перейменуйте та перемістіть у відповідну папку в `public/images/`
